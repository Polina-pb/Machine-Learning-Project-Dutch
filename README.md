# Датирование исторических текстов на нидерландском языке (XIII-XIX вв.)

## Цель проекта
Создать и сравнить несколько моделей, которые бы классифицировали тексты на нидерландском языке в зависимости от века, в котором они изданы (или написаны - в случае более позднего издания книги с сохранением особенностей орфографии на момент ее написания).

## Задачи проекта
1. Собрать репрезентативный корпус текстов на нидерландском языке за XIII-XIX вв., минимизировать дисбаланс классов за счет сбора материала из различных источников.
2. Периодизировать историю развития нидерландского языка для корректного определения таргета.
3. Предобработать тексты и сопоставить их с имеющимися по ним метаданными.
4. Выявить потенциальные признаки, которые могли бы быть использованы при обучении моделей, по итогам изучения научных статей и анализа текстов.
5. Произвести подсчет метрик по признакам и отобрать релеватные для обучения моделей признаки (признаки, которые коррелируют с таргетом и при этом не являются мультиколлинеарными).
6. Обучить несколько классических линейных и нелинейных моделей и оценить результаты их работы.
7. С учетом полученных результатов вернуться к этапам 1-6 и усовершенствовать материалы для обучения моделей и параметры моделей.
8. Подготовить тексты для их передачи в нейронную сеть и обучить нейронную сеть на базе предобученных моделей Bert (двух исторических моделей и одной современной).
9. Оценить результаты работы нейросети, сравнить их с результатами работы классических моделей, сделать вывод о наиболее оптимальном пути для датирования исторических текстов на нидерландском языке.

## Результаты работы
* Классические модели показали неплохие результаты (выше 0,80 по precision, recall и f1-score), в особенности нелинейные модели (K ближайших соседей, Дерево решений) и амсамбли (стекинг и бустинг).
* Показали нейронной сети немного выше, но время их работы значительно дольше, чем время работы классических моделей, при этом современной модели Bert (Bertje) требуется больше эпох для обучения, чем историческим моделям. 

*Ограничения результатов работы и пути для совершенствования моделей:*
* необходимость обеспечения большего количества примеров первого класса (1200-1500 гг., всего около 80 текстов) для более взвешенной оценки моделей;
* целесообразность обучения нейронной сети в течение большего количества эпох для оптимизации модели (в рамках проекта было заложено всего 2 эпохи из-за ограничений работы GPU в Google Colab);
* как показали итоги работы как классических моделей, так и нейронной сети, в рамках совершенствования моделей может быть целесообразно дальнейшее уточнение границ второго класса (1500-1700 гг.) для обеспечения большей точности его предсказания.
